Welcome to TeacherManager

Manage classroom attendance, student profiles, and schedules with ease.

A secure, role-based PWA for teachers, administrators, parents, and students.

Core Features

📋

Attendance Tracking

Mark attendance in real-time, edit records, and view history with ease.

👤

Student Profiles

Manage student information, enrollment dates, parent contacts, and more.

📅

Class Management

Create and manage classes, schedules, and student enrollment.

🔔

Smart Alerts

Automatic notifications for excessive absences and important events.

📊

Reports & Analytics

Generate PDF reports and view dashboards with attendance analytics.

🔐

Role-Based Access

Admin, Teacher, Parent, and Student roles with fine-grained permissions.

User Roles

👨‍🏫 Teacher

  • Generate class reports

Technology Stack

  • Backend: FastAPI (Python 3.12) with async SQLAlchemy
  • Frontend: Jinja2 templates + HTML5 + CSS3 + Service Workers (PWA)
  • Database: PostgreSQL with Alembic migrations
  • Caching: Redis for sessions and performance
  • Auth: JWT tokens with RBAC (Role-Based Access Control)
  • APM: Sentry for error tracking and monitoring
  • Reports: WeasyPrint for PDF generation
